Welcome to the GWT Eclipse Plugin update site

This web page provides automatic distribution and updates for the GWT Eclipse Plugin.

Plugin installation

To install the plugin:

  1. In Eclipse, click on Help -> Install New Software... -> Add...
  2. (The "Add Repository" dialog will appear)
  3. For "Name", enter "GWT Plugin" or something similar
  4. For "Location", fill-in the URL of this web page
